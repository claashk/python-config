Content Handler Interface¶

The ContentHandler defines the interface for classes designed to obtain content from reader objects. As such ContentHandler can be regarded as an abstract base class. Examples of classes implementing this interface are Validator or XmlWriter.

ContentHandler Interface¶

class schema.content_handler.ContentHandler[source]¶

Processes content obtained from a reader

ContentHandler derived objects are similar to xml.sax content handlers. They are intended to provide a defined interface for readers to pass in data and to distribute or process this data. Writers or validators are classes typically derived from this class.

This is a pure base class defining the interface. For possible implementations refer to * Validator * XmlWriter
